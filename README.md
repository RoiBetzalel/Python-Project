# Minecraft Speedrun Data Analysis with Python

## 📖 Overview

This project analyzes Minecraft speedrun data from Couriway using Python and Jupyter Notebook.

The project focuses on cleaning, exploring, and visualizing gameplay data to identify patterns in speedrun performance, run progression, and player statistics.

---

## Dataset

The dataset contains information from Minecraft speedruns, including:

- Date Played
- Seed
- Iron Source
- Enter Type
- Gold Source
- Nether Entry
- Bastion
- Fortress
- Nether Exit
- Stronghold
- End
- In-Game Time (IGT)
- Blaze Rods
- Death Count
- Blaze Kills
- Enderman Kills
- Iron Golem Kills
- Bastion Type
- End Fight Type
- Frame Eyes
